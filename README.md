# HR Role in the AI Transformation

Interactive slideshow presentation exploring the evolving role of HR in the age of AI.

## Overview

A single-page, self-contained presentation built for HR leadership, covering:

1. **Opening** — HR as the Owner of the AI Workforce
2. **The Shift** — Traditional HR vs. HR in the AI era
3. **Strategic Pillars** — Workforce Planning, Role Redesign, Change & Adoption, Governance
4. **RACI Model** — Hybrid human-AI work model with a credit risk analysis example
5. **Roadmap** — 180-day execution plan with KPIs
6. **Closing** — Call to action

## Features

- Full keyboard, mouse, touch/swipe navigation
- RTL (Hebrew) layout with Heebo font
- Responsive design — works on desktop, tablet, and mobile
- Progress bar and dot navigation
- White + red (Bank Hapoalim) color scheme
